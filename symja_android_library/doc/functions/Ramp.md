## Ramp

```
Ramp(z)
```

> The `Ramp` function is a unary real function, whose graph is shaped like a ramp.  
 
See
* [Wikipedia - Ramp function](https://en.wikipedia.org/wiki/Ramp_function)

### Examples

```
>> PiecewiseExpand(Ramp(x)) 
Piecewise({{x,x>=0},0})
```
