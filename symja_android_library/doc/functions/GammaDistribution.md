## GammaDistribution

```
GammaDistribution(a,b)
```

> returns a gamma distribution.
    
See:  
* [Wikipedia - Gamma distribution](https://en.wikipedia.org/wiki/Gamma_distribution)

 
### Examples

The variance of the gamma distribution is

```
>> Variance(GammaDistribution(n, m)) 
m^2*n
```
