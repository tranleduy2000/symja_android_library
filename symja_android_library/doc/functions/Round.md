## Round

```
Round(expr)
```

> round a given `expr` to nearest integer.

See
* [Wikipedia - Floor and ceiling functions](https://en.wikipedia.org/wiki/Floor_and_ceiling_functions)

### Examples

```
>> Round(-3.6)
-4
```
