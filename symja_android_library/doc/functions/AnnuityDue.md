## AnnuityDue

```
AnnuityDue(p, t)
```

or 

```
AnnuityDue(p, t, q)
```

> returns an annuity due object.
  

See:
* [Wikipedia - Annuity-due](https://en.wikipedia.org/wiki/Annuity#Annuity-due)
 
### Examples

```
>> TimeValue(AnnuityDue(100, 12), 0.1, 0) 
749.5061
```
