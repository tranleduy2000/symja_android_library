## FractionalPart

```
FractionalPart(number)
```

> get the fractional part of a `number`. 

See
* [Wikipedia - Fractional part](https://en.wikipedia.org/wiki/Fractional_part)

### Examples

```
>> FractionalPart(1.5)
0.5  
```
