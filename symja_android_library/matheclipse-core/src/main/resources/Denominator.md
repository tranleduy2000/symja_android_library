## Denominator

```
Denominator(expr)
```

> gives the denominator in `expr`. Denominator collects expressions with negative exponents.

See
* [Wikipedia - Fraction (mathematics)](https://en.wikipedia.org/wiki/Fraction_(mathematics))

### Examples

```
>> Denominator(a / b)
b
>> Denominator(2 / 3)
3
>> Denominator(a + b)
1
```
