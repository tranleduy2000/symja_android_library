## ToCharacterCode

```
ToCharacterCode(string)
```

> converts `string` into a list of corresponding integer character codes.

 
### Examples

```
>> ToCharacterCode("ABCD abcd")
{65,66,67,68,32,97,98,99,100}

>> "a-3" // ToCharacterCode
{97,45,51}
```
