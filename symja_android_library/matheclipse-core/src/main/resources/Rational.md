## Rational

```
Rational
```

> is the head of rational numbers.

```
Rational(a, b)
```

> constructs the rational number `a / b`.

See 
* [Wikipedia - Rational number](https://en.wikipedia.org/wiki/Rational_number)

### Examples

```
>> Head(1/2)
Rational

>> Rational(1, 2)
1/2

>> -2/3
-2/3
```
