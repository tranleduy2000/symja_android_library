## Association

```
Association(list-of-rules) 
```

> create a `key->value` association map from the `list-of-rules`.
 
### Examples

```
>> Association({ahey->avalue, bkey->bvalue, ckey->cvalue}) 
<|akey->avalue,bkey->bvalue,ckey->cvalue|> 
```
